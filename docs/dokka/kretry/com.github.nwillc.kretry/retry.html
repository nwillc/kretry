retry

inline fun <R> retry(config: Config<R> = Config(), block: () -> R): R

A retrying extension that will retry a given function based on a configuration. The block is run repeatedly until succeeding based on the config predicate, or until the config attempts exceeded.

Parameters

config - The retrying Config.

block - The function to retry.

Exceptions

RetryExceededException - If the config attempts exceeded.
