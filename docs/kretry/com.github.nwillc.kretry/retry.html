retry

fun <C : Any, T> C.retry(config: Config<T> = Config(), func: C.() -> T): T

A retrying extension that will retry a given function based on a configuration. The func is run repeatedly until succeeding based on the config predicate, or until the config attempts exceeded.

Parameters

config - The retrying Config.

func - The function to retry.

Exceptions

RetryExceededException - If the config attempts exceeded.
